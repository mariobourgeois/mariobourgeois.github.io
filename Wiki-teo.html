Objectifs:

En s'inspirant des sites existants:
  • Réaliser une maquette d'un wiki, cette maquette devra comprendre les affichages PC, tablette et mobile.
  • Développer en HTML / CSS / JavaScript la partie front-end du site à partir de la maquette précédemment créée.
  • Utiliser un éditeur de code de votre choix (Visual studio code conseillé, sublim text, le bloc note, etc....)
  • utiliser le langage PHP pour la partie back-end.
  • Créer base de données.

Compétences:

Les compétences utilisées sont:
  • Maquetter une application.
  • Réaliser une interface utilisateur web statique et adaptable.

Langages:

Les langages utilisées sont:
  • HTML/CSS

Logiciels/Outils:

Les logiciels et outils utilisés sont:
  • Visual Studio Code.
  • Mockflow.
  • Bootstrap.
  • Git/Github.
